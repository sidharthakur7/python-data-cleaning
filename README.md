# Python Data Cleaning Tool

## 📌 Description
A simple Python tool to clean and preprocess CSV data by removing missing values, duplicates, and formatting column names.

## ⚙️ Features
- Removes missing values
- Removes duplicate rows
- Standardizes column names

## 🛠️ Technologies Used
- Python
- Pandas

## 📂 Project Structure
- data_cleaning.py → main script
- data.csv → raw input data
- cleaned_data.csv → cleaned output data

## 🚀 How to Run

1. Install pandas:
   pip install pandas

2. Run the script:
   python data_cleaning.py

## 📊 Example

Input data may contain:
- Missing values
- Duplicate rows

Output:
- Clean, structured dataset ready for analysis

Cleaned CSV file generated successfully
